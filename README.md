## 计划迭代
1. 通过一个按键实现记录多个时间点（包括起床、出门、到达工位、休息/继续、离开工位、下班到家、睡觉）
2. 统计上班期间的休息时长（按休息开始/继续计时，按继续暂停计时，按离开工位结束计时并记录）
3. 生成该工作日上班时间情况海报，并可以保存到本地（点击“生成今日海报”可以预览，再点击保存进行保存）
